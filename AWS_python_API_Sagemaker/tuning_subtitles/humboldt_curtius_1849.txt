Ernst Curtius
Ernst Curtius eine Lebensbild in Brief.
neu Ausgabe von Friedrich Curtius.
Erster Band mit zwei Bildnis.
zweit Auflage.
Verlag Karl Curtius Berlin 1913
Alexander von Humboldt in eine zum Geburtstag von Ernst Curtius 2. September 1849 von d Prinzessin von Preußen geschenkt Album:
"wie d Vogel auf d Felsen über d schäumend Wasserfall (d letzte d Aturen, die sie so anmutig besingen) sein ich übrig bleiben aus d Schiffbruch d alt Geschlecht.
wenn ihr Blick, teuer Curtius, sich weiden können an d griechisch Landschaft, an d innig Verschmelzung d Starren und Flüssige, d mit Zypresse oder Oleander geschmückt oder felsig, luftgefärbt Ufer, d wellenschlagend, lichtwechselnd, glanzvoll Meer, wenn d ewig, unwandelbar Größe d frei Natur, in welche d hingescheiden Größe von Helleas sich spiegeln, ihr regsam Gemüt und ihr Sprache veredeln, werden ich, d Wandernde, nur zuteil, an namenlos Fluß, in d dicht und wild Forst d Orinoco, zwischen schneebedeckt Feuerberg, in d endelos Grasflur und Steppe d Jrtysch und Obi zu verweilen.
einsam werden ich mich fühlen, eine d letzte von d alt Geschlecht, haben Freundschaft nicht, die alle lindernd, ich ihr wohltätig, süß Gabe spenden.
auf d klein, laubbekränzt Hügel, wo Geist und hold Anmut walten, sein ich an eine festlich, von d edel still gefeiert Tag d Freude werden, d tieffühlend Naturmaler von Naxos diese wenige Zeile (in ungezähmt Freiheit cursibus obliquis fluentes) aus gerade Sinn dankbar und liebevoll zu widmen."
beziehen sich auf eine Gedicht, welche Humboldt in d "Ansicht d Natur" veröffentlichen haben.
beziehen sich auf Humboldt Schreibweise.
seine Zeile gehen von links nach rechts in d Höhe.
